 🚗 Motor Vehicle Insurance Management System

# 📖 Project Overview
The Motor Vehicle Insurance Management System is a MySQL-based database project developed using MySQL Workbench. It manages customer, vehicle, insurance policy, premium payment, claim, and agent information using a relational database.

# 🎯 Objectives
- Manage customer records
- Maintain vehicle information
- Issue insurance policies
- Track premium payments
- Process insurance claims
- Manage insurance agents
